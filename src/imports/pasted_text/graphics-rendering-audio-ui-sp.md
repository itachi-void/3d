=========================================================
ADAPTIVE QUALITY
=========================================================

If FPS drops:

Reduce particles.

Reduce shadow resolution.

Reduce post processing.

Reduce reflections.

Reduce volumetric quality.

Reduce animation complexity.

If FPS recovers:

Gradually restore quality.

Never switch instantly.
