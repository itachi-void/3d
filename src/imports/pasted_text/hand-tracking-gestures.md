=========================================================
HAND TRACKING SYSTEM
=========================================================

The entire application must be controlled using hand gestures.

The user should never need a mouse or keyboard after the application starts.

Use the highest quality hand tracking available.

Preferred:

- MediaPipe Tasks Vision
- MediaPipe Hands
- Custom gesture classifier if needed

Track:

• Left Hand
• Right Hand
• Wrist
• Palm Center
• Palm Rotation
• Palm Normal
• Every Finger
• Every Finger Joint
• Fingertips
• Hand Velocity
• Hand Direction
• Distance Between Hands
• Hand Confidence
• Gesture Confidence

=========================================================
TRACKING QUALITY
=========================================================

Requirements:

No jitter.

No sudden jumps.

Use interpolation.

Use exponential smoothing.

Use damping.

Use prediction to reduce latency.

Ignore accidental detections.

Use confidence thresholds.

Switch gestures only after they remain stable.

Prevent flickering.
