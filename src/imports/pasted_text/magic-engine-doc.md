=========================================================
MAGIC ENGINE
=========================================================

The application should simulate a living magical world.

Magic is not a visual effect.

Magic is a complete interaction system.

Every spell affects:

• Particles
• Physics
• Environment
• Camera
• Lighting
• Audio
• Weather
• Objects
• NPCs

All magical actions must feel powerful.

Nothing should look fake.

=========================================================
ENERGY SYSTEM
=========================================================

The player has unlimited magical energy.

Every spell generates visible magical energy.

Energy appears as:

• Floating particles
• Energy ribbons
• Sparks
• Smoke
• Heat distortion
• Dynamic light

Energy color changes according to current world.

Temple:
Gold

Castle:
Purple

Sci-Fi:
Blue

Space:
White

Ocean:
Cyan

=========================================================
TELEKINESIS
=========================================================

Point at an object.

Object highlights.

Pinch to grab.

Move hand:

Object follows naturally.

Object should have inertia.

Release:

Object keeps momentum.

Object collides with environment.

Can destroy other objects.

=========================================================

Leaves fly.
